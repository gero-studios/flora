## Environment Variables

| Variable | Description |
|----------|-------------|
| `CMD_DEMO` | Set to `1` to hide internal team badges (e.g. `co`) from the CLI header. Useful for demos and screenshots. Usage: `CMD_DEMO=1 cmd` |
